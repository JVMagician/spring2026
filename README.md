Before using the source code below, make sure you have completed the following steps (as outlined in Canvas):

- Create a new project in IntelliJ
- Name the project: ```pgccdownloaderapp```
- Build and run the project to confirm everything is working correctly.
- You should see the default JavaFX application window.
- Verify functionality
  - Ensure the app launches without errors before continuing.

## Refactor the default files
- Ensure to right click (on the file) -> refactor -> Rename the following files:
  - hello-view.fxml → ```downloader-view.fxml```
  - HelloApplication.java → ```DownloaderApplication.java```
  - HelloController.java → ```DownloaderController.java```

# MyCalcus Project Instructions

## 1. Download the Project
- Download the compressed file [`myCalcus.zip`](https://github.com/JVMagician/spring2026/blob/main/myCalcus.zip) to your local computer.
## 2. Extract the Files
- Unzip the downloaded folder to a location of your choice.
## 3. Open the Project in IntelliJ IDEA
- Launch IntelliJ IDEA.
- Open the unzipped folder containing the `myCalcus` source code.
## 4. Run the Project
- Run the code to see the output of the partially completed calculator application.
## 5. Complete the Calculator Features
- Implement all missing button functionalities:
  - Refer to the snapshot below to identify missing buttons in **SceneBuilder**.
  - Implement the event-handling logic in `CalcusController.java`.
  - Test the application to ensure all buttons work correctly mathematically.
## 6. Submit Your Work
- Upload your completed project to your own GitHub repository.
- Share the link Github link with me.
